# vimrcs
takubokudori's vimrc files(for Japanese language)

## 内容物

- .vimrc
- .gvimrc
- .ideavimrc
- .vsvimrc

## インストール

### Windows
ユーザ環境変数「HOME」にvimrcを置くフォルダを設定し，以下の手順で設定します．

```
cd %HOME%
git clone https://github.com/takubokudori/vimrcs
cd vimrcs
install.bat
```

### Linux
```
cd $HOME
git clone https://github.com/takubokudori/vimrcs
cd vimrcs
chmod +x install.sh
./install.sh
```
